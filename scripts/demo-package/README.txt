LittleTank Demo

How to run on Windows:
1. Extract this zip file.
2. First try double-clicking "LittleTank Demo.html".
3. If the browser shows a blank page because of local file restrictions, double-click "Start LittleTank Demo.cmd" instead.
4. The .cmd fallback opens a local browser window automatically. Keep its command window open while playing, and close it to stop the local demo server.

This package includes the current built game files and the current designer data workbooks under designer-data/.
It does not require Vite, npm install, or a manually started web server.
